Chipper is a project made by Hadekha Erfadila Fitra, this twitter-like project is made for educational purposes.

This website implements / uses:

  • angular as typescript front-end framework

  • golang as backend language

  • mongodb atlas as database as services

  • ngrx/store and ngrx/effects which is redux-like state management for angular

  • websocket to establish realtime connection

  • json web token as authentication method

  • google's captcha v4 as anti-bot screening

  • email verification
